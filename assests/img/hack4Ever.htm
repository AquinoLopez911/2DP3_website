Converting a string to pig latin involves moving the first letter of each word to the end of it, then adding "ay" to the end of the word.

  • Create a class called 'PigLatin'
  • Create a method 'convert' which takes a string parameter s
  • Move the first letter of each word to the end of it, then add 'ay' to the end of the word.
  • Leave punctuation marks and numbers untouched.
  • One word words should just add 'ay' to the end. 'I'->'Iay'
  • Words that start and end with the same letter should NOT double up. 'spaces'->'pacesay' not 'pacessay'
  • Words that start with 'a' should NOT double up. 'art'->'rtay' not 'rtaay'
